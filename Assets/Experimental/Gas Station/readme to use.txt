~~~~~~For .U3D, .X, and .OBJ, .FBX, .MS3D version of the model
1) Extract the .U3D, .X, and .OBJ file to your preferred directory
2) Extract the textures within the root folder of the compressed archive to the same directory as you used above
3) DO NOT EXTRACT THE CONTENTS OF THE "GAS STATION" FOLDER - the folder and it's contents are not used by the file types described above!

~~~~~FOR .CSM version of the model
1) Extract the "gas station" folder to the textures directory of the Cartography Shop installation.
2) Run Cartography Shop
3) Open .CSM file =)
